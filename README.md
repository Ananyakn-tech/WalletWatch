# WalletWatch

## Expense Tracking Dashboard

WalletWatch is a frontend web application designed to help users track and analyze their daily spending habits through a clean and interactive dashboard interface.

## Features

- Add expense transactions with category selection  
- Track daily, weekly, and monthly expenses  
- Transaction history management  
- Category-wise doughnut chart visualization  
- LocalStorage-based data persistence  
- Responsive modern UI dashboard  

## Technologies Used

- HTML  
- CSS  
- JavaScript  
- Chart.js  

## Project Purpose

This project was built to practice frontend development concepts such as DOM manipulation,data visualization and client-side storage handling.
